= Dracula Cursor Theme

image:https://img.shields.io/badge/license-GPLv3-green.svg?style=flat[license]

A https://github.com/dracula/dracula-theme[dracula]-themed cursor set forked from https://github.com/rtgiskard/bibata_cursor[rtgiskard/bibata_cursor],
which is based on the original https://github.com/ful1e5/Bibata_Cursor[Bibata_Cursor] project.
This theme is specially adapted for the dracula color scheme and provides cursors for both
hyprcursor and Xcursor environments with the iconic dark background and purple accents.


== Build

- Prerequisites

* Python `>= 3.12`
* https://wiki.gnome.org/Projects/LibRsvg[librsvg]: generating PNG files (Xcursor only)
* https://xorg.freedesktop.org[xorg-xcursorgen]: generating Xcursor from PNG and related (Xcursor only)

- Quick start
+
```bash
# 1. install the dependencies
# Arch Linux:  pacman -Syu python librsvg xorg-xcursorgen

# 2. clone
git clone https://github.com/diljitht/dracula_cursor_dracula.git

# 3. build
cd bibata_cursor_dracula
python src/cursor_utils.py --hypr --x11 --out-dir ./out

# 4. copy to user local icon theme
cp -r ./out/dracula-cursor-theme ~/.local/share/icons/
```

- The process of creating the cursor themes:

1. Recolor SVG files according to the dracula color scheme in `config/render.json`
2. Parse cursor settings from `config/build.toml`
3. Generate cursor config for hyprcursor or Xcursor
4. Render SVG files to PNG files (Xcursor only)
5. Build cursor themes from SVG (hyprcursor) or PNG (Xcursor) files.


== Examples

```bash
# Run the utils to check the help:
python src/cursor_utils.py -h

# Generate hyprcursor to `./out`:
python src/cursor_utils.py --hypr --out-dir ./out

# Generate both hyprcursor and Xcursor themes:
python src/cursor_utils.py --hypr --x11 --out-dir ./out
```

== Dracula Color Scheme

This theme uses the classic dracula color palette:

* **Background**: `#282A26` (dark background)
* **Primary**: `#BD93F9` (dracula purple)
* **Red**: `#FF5555`, **Pink**: `#FF79C6`, **Green**: `#50FA7B`
* **Cyan**: `#8BE9FD`, **Yellow**: `#F1FA8C`, **White**: `#F8F8F2`

== Customization

=== Customize Sizes

Adjust `config/build.toml: cursor_defaults.x11_sizes` to include only the sizes you need to reduce the cursor theme size.
For hyprcursor, themes are rendered as vector images with no need for specific sizes.

=== Customize Colors

Modify the color mappings in `config/render.json` to customize the dracula palette. The file contains mappings from original SVG colors to their dracula equivalents.

This dracula cursor theme is a standalone implementation with no compatibility guarantees with upstream projects.
